# Snake CLI

A simple terminal snake game written in Rust using crossterm

## Installation

Install Rust by following the official installation guide:
https://doc.rust-lang.org/cargo/getting-started/installation.html

Then install the game by running:
```bash
cargo install snake-cli
```

## Usage

After installing, just run:
```bash
snake-cli
```

## Controls

| Key | Action |
|:---:|--------|
| W / ↑ | Move Up |
| A / ← | Move Left |
| S / ↓ | Move Down |
| D / → | Move Right |
| Q | Quit |
| R | Restart Game |
